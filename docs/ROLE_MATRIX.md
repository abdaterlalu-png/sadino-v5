# Role SADINO

- Creator: full superadmin terhadap seluruh aplikasi, data, user, role, desain, import, export, backup, dan konfigurasi, dengan guard self-delete, last Creator, audit, konfirmasi, dan backup.
- Finance Manager: validasi/final review financial, target, AR, AP, cashflow, P&L, asset, dan laporan.
- Accountant: input, koreksi, upload Excel, mapping, AR/AP, rekonsiliasi, aset, pajak, dan draft laporan.
- Director: view, drilldown, export, target, laporan, dan AI analysis; tidak mengubah sumber data secara default.
